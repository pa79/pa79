## Highlighted Projects

### [Macro Regime Feature Platform](https://github.com/pa79/macro-regime-feature-platform)
Production-style Python and DuckDB pipeline that ingests macro and market data, validates it, materializes a gold-layer feature store, and serves a Streamlit dashboard for regime and risk monitoring.

### [LendingClub PD Scorecard](https://github.com/pa79/lendingclub-pd-scorecard)
Retail credit risk scorecard built on 1M+ historical loans with time-based validation, WOE transformation, logistic regression, calibration checks, score bands, and underwriting policy outputs.

### [Causal Impact of Gig Worker Reclassification](https://github.com/pa79/Causal-Inference-Prop-22-)
End-to-end causal analysis pipeline using difference-in-differences, event studies, and synthetic control to estimate policy effects on employment, earnings, and pricing.

### [Forecasting U.S. GDP](https://github.com/pa79/Forecasting-US-GDP)
Macro forecasting project using economic indicators and time-series regression to estimate near-term GDP growth under changing business-cycle conditions.

### [Loan Default Prediction](https://github.com/pa79/Predicting-loan-default)
Credit risk modeling workflow that aggregates transactional banking data into borrower-level features for default prediction and model evaluation.
